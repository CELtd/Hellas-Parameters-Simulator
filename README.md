# Hellas Fraud Game Simulator (Webapp)

Interactive web application for exploring the economic dynamics of the Hellas Protocol's fraud game mechanism. Use it to analyze equilibrium behavior, run Monte Carlo simulations, and visualize attack vectors.

## Features
- Real-time equilibrium analysis with sensitivity charts
- Monte Carlo simulations with configurable parameters
- Attack vector impact visualization
- Modern UI with Tailwind CSS and Framer Motion

## Getting Started

### Prerequisites
- Node.js 18+
- npm (or yarn)

### Install & Run
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
npm run preview
```

## Deployment (Vercel)
- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
